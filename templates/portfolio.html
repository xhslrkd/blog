Comfort Women Interactive Video

Technology Summary

Processing(Java-based software sketchbook for visual effects)

Description

This was for my English class project. Thinking about how to catch people's attention, I have decided to emulate a comfort women memorial. The memorial consists of a girl sitting on a chair next to an empty chair that expresses sympathy for them. With this in mind, my program runs an informative video of them when a person sits in front of a computer monitor with a flower in a specific color; the color triggers the video when it is spotted on a certain area using a simple color detection technique.

Technology Summary

Laravel PHP HTML5 CSS3 Javascript

Description

For my Computer Science Practicum course, my team, which consisted of three people, made a mock property management website. It is an app that allows for a smooth interaction between landlords and tenants. People could sign up either as a landlord or a tenant. Functionalities include landlords requesting money and maintenance requests for tenants as well as simple password resetting through email and the website's contact information. This project was built with Laravel, a PHP-based web framework.
